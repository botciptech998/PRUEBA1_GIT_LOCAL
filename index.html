Esta es una pagina de prueba para practicar el uso de git en un entorno local.

Aqui podras encontrar informacion sobre como utilizar git y sus comandos basicos.

Comandos basicos de git

  • git init: Inicializa un nuevo repositorio git.
  • git add: Agrega archivos al area de preparacion.
  • git commit: Guarda los cambios en el repositorio.
  • git status: Muestra el estado actual del repositorio.
  • git log: Muestra el historial de commits.
